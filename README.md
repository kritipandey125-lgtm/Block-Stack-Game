# 🧱 Block Stack Game

A simple and fun block stacking game where players stack moving blocks to build the tallest tower. Test your timing and precision to achieve the highest score!

## 🚀 Features
- Easy-to-play gameplay
- Score tracking
- Responsive design
- Restart game functionality

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🎮 How to Play
- Click or tap to drop the moving block.
- Stack blocks as accurately as possible.
- The game ends when a block misses the stack.
